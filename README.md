# PWA2IPA
A tool which turns Progressive Web Apps into actual apps for iOS systems.
# how to use
1. fork this repo
2. replace "ucc-icon.png" with your icon. (if file names are different please change it in the buildipa.yml file.)
3. replace all files in folder "www" with your html files.
4. check actions page and click the latest build
5. wait for it to build
6. check artifacts when finished and download the zip
   
# IMPORTANT NOTE
The output IPA file is **NOT** signed so you need to either sign it using a signing tool or use a paid developer account.

# my recommendations
1. 3utools (https://3u.com)
2. sideloadly (https://sideloadly.io)
3. a paid dev account
